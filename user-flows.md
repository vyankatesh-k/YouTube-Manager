# User Flows – AIM 2026

## Daily Usage Flow

1. User logs in
2. Views today’s date
3. Plans activities for the day
4. Executes tasks
5. Marks completed activities
6. Adds reflection at end of day

---

## Weekly Review Flow

1. User opens weekly summary
2. Reviews execution consistency
3. Identifies missed or partial days
4. Adjusts future planning accordingly
