## Objective

The objective of the MVP is to provide creators with a clear, simple system to:
- Plan daily activities
- Track execution
- Review weekly consistency

---

## In Scope (Phase 1)

### 1. User Authentication
- Basic email/password authentication
- Secure session management

### 2. Creator Profile
- Channel name
- Niche
- Weekly content target

### 3. Activity Management
- Predefined activity types (e.g., Script Writing, Editing, Shooting)
- Ability to add custom activities

### 4. Daily Activity Logs
- Daily planning of activities
- Marking activities as completed or missed
- Time spent tracking
- End-of-day reflection

### 5. Weekly Summary View
- 7-day execution overview
- Consistency indicators
- Missed and partial days highlighted

---

## Out of Scope (Explicitly Excluded)

- YouTube analytics integration
- AI-based recommendations
- Automation or scheduling
- Team or collaboration features
- Monetization features

The MVP prioritizes clarity and discipline over feature richness.
